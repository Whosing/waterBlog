    播放清單

  • 將播放清單儲存媒體庫不論是您先前建立的播放清單、其他創作者建立的播放清單，或是您加入稍後觀看清單中的影片，您都可以輕鬆地將這些內容新增到媒體庫中，日後就能快速地找到這些內容，並盡情觀賞。


  • 您可以把自己的播放清單新增到媒體庫，然後設為公開，這樣其他使用者就能新增這些播放清單到他們自己的媒體庫中。
  • 如果不想讓別人看到您的播放清單，只要調整隱私設定即可。完成後，您還是能在自己的頻道中看到您已儲存的播放清單，但其他人則不會再看到。

共筆系統

  • 網址：https://paper.dropbox.com/
  • 說明：在電腦上如果想要紀錄一些簡單的資訊，應該很少有人會選擇新增一個 word 檔案，通常比較可能會新建一個純文字檔案。會選擇一個多人共同編輯的純文字檔案。而它不單純只是純文字檔，也支援簡單的格式，可以簡單的製作一個看起來還不錯的文件。在線上工作，常常需要整理文件、列出待辦事項、徵求幫手、提供教學文件、記錄備忘等等。

程式碼管理

  • 網址：https://github.com
  • 說明：很多人都有這樣的煩惱：程式碼修過好幾次以後，早已認不得最新的是哪個版本，最舊的是哪個版本。為了解決這個問題，Linus Torvalds開發了 git 作為管理程式碼的工具。git 是一個很好用的分散式程式碼版本管理工具。而 github 則是使用 git 的社交網站。每個人都可以有自己的程式碼，你可以在上面複製，或是協助修改別人的程式碼。
  • 相關連結：https://github.com/Whosing/
